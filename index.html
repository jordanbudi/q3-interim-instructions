Task 9: Implement the use of a loop (a while loop OR a for loop).

Loops allow us to create programs that executes large amounts of actions with very little lines of code. We've done a number of examples in class that lets you combine your knowledge of variables and loops to replicate actions. A simple exmaple would be to use a loop to create a row of shapes (like squares or rectangles) by only writing one line for the shape and then wrapping it in a while loop or for loop.

Pro tip: be careful of creating an infite loop (a loop that goes on forever and never ends). Infinite loops will crash your browser. The most common cause of this is that you forgot to increment your variable so that the loop exits.
